# AnkiFlow Docs

Welcome to the project documentation for AnkiFlow.

AnkiFlow is a Nuxt-based app for creating Anki cards from words with an LLM-assisted review flow.

## Documentation Map

- [Project Plan](./project-plan.md)
- [Architecture Notes](./architecture.md)
- [Decisions Log](./decisions.md)
- [Setup Notes](./setup.md)

## Project Summary

The main user flow is:

1. Enter a word.
2. Generate card content.
3. Review and edit the result.
4. Confirm and add the note to Anki.
